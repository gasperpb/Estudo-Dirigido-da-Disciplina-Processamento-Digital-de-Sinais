# 🎓 Estudo Dirigido - Processamento Digital de Sinais (PDS)

> **Seminário Completo de Processamento Digital de Sinais**
> 
> Este projeto apresenta um estudo dirigido completo sobre PDS, estruturado como um seminário acadêmico com teoria, prática e recursos audiovisuais para aprendizagem progressiva.

---

## 📚 Sobre o Projeto

Este seminário foi desenvolvido para fornecer uma compreensão abrangente dos conceitos fundamentais em **Processamento Digital de Sinais (PDS)**. Cada módulo inclui:

- 📖 **Resumos teóricos** detalhados
- 💻 **Simulações práticas** em Python/Jupyter
- 🎥 **Vídeos educacionais** selecionados
- 🔗 **Links diretos** para todos os recursos

## 🎯 Como Seguir o Seminário

### 📖 **Passo 1: Preparação**
1. Clone ou baixe este repositório
2. Instale as dependências Python necessárias:
   ```bash
   pip install numpy matplotlib scipy jupyter
   ```
3. Abra o Jupyter Notebook para executar simulações

### 🗺️ **Passo 2: Roteiro de Estudos**

#### **Semana 1-2: Fundamentos Matemáticos**
- [ ] 📚 Leia o [Resumo da Transformada-Z](./Parte%201%20A%20transformada-z/Resumo.md)
- [ ] 🎥 Assista aos [Vídeos da Transformada-Z](./Parte%201%20A%20transformada-z/Videos.md)
- [ ] 💻 Execute as [Simulações da Transformada-Z](./Parte%201%20A%20transformada-z/Simulacoes.ipynb)

#### **Semana 3-4: Análise Espectral**
- [ ] 📚 Leia o [Resumo da DFT](./Parte%202%20DFT/Resumo.md)
- [ ] 🎥 Assista aos [Vídeos da DFT](./Parte%202%20DFT/Videos.md)
- [ ] 💻 Execute as [Simulações da DFT](./Parte%202%20DFT/Simulacoes.ipynb)

#### **Semana 5-7: Filtragem Digital**
- [ ] 📚 Estude [Filtros Discretos](./Parte%203%20Filtros%20Discretos/Resumo.md)
- [ ] 📚 Aprofunde em [Filtros FIR](./Parte%204%20Filtros%20FIR/Resumo.md)
- [ ] 📚 Domine [Filtros IIR](./Parte%205%20Filtros%20IIR/Resumo.md)
- [ ] 💻 Pratique com todas as simulações de filtros

#### **Semana 8: Aplicações Práticas**
- [ ] 📚 Explore [Aplicações em Comunicações](./Parte%206%20Aplicações%20em%20Comunicações/Resumo.md)
- [ ] 🎥 Assista aos [Vídeos de Comunicações](./Parte%206%20Aplicações%20em%20Comunicações/Videos.md)
- [ ] 💻 Execute as [Simulações de Comunicações](./Parte%206%20Aplicações%20em%20Comunicações/Simulacoes.ipynb)
